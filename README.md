## üìù Project Description

This repository serves as a portfolio showcasing projects focused on **Time Series Analysis** and **Exploratory Data Analysis (EDA)**. The projects demonstrate proficiency in handling time-dependent data, identifying patterns, and performing statistical analysis to extract meaningful insights.

The main goal is to apply Python's robust data science ecosystem to solve real-world problems across different domains, such as climate, sales, and energy consumption.

<!-- Features and Application Demonstration -->
## ‚ú® Features and Application Demonstration

The repository consists of Jupyter Notebooks, each detailing a specific analysis project. The table below summarizes the focus, objectives, and key findings of each project:

| Project | Primary Focus | Key Objectives | Demonstration (Notebook) |
| :--- | :--- | :--- | :--- |
| **Climate Time Series Analysis** | Analysis of daily climate data (temperature, humidity, wind speed, pressure) for New Delhi. | Identify seasonality, perform data cleaning, and visualize the time-dependent behavior of climate variables. | [`ClimateTimeSeriesAnalysis.ipynb`](https://github.com/lucaspn017/time_series_analysis/blob/main/ClimateTimeSeriesAnalysis.ipynb) |
| **Sales Data Inspection** | Exploratory Data Analysis (EDA) and data cleaning on a Superstore sales dataset. | Analyze sales distribution by region and category, and apply clustering (K-Means) to identify customer segments. | [`Inpe√ß√£o_dados_de_vendas.ipynb`](https://github.com/lucaspn017/time_series_analysis/blob/main/Inpe%C3%A7%C3%A3o_dados_de_vendas.ipynb) |
| **Energy Consumption Analysis** | Time series analysis of energy consumption data, focusing on peak and off-peak hours. | Separate consumption values into peak and off-peak periods and calculate excess reactive energy. | [`Time_Series_Analysis.ipynb`](https://github.com/lucaspn017/time_series_analysis/blob/main/Time_Series_Analysis.ipynb) |

**Demonstration:** To view the analyses, simply click on the notebook links above. GitHub renders the full content, including code and visualizations, directly in the browser.

<!-- Project Access -->
## üíª Project Access

The source code and all notebooks are available in this repository:

[**Access GitHub Repository**](https://github.com/lucaspn017/time_series_analysis)

To run the projects locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/lucaspn017/time_series_analysis.git
    ```
2.  **Install dependencies:**
    ```bash
    # It is recommended to create a virtual environment first
    pip install pandas numpy matplotlib seaborn scikit-learn jupyter
    # For the energy analysis notebook, you may need to install openpyxl:
    pip install openpyxl
    ```
3.  **Start Jupyter Lab/Notebook:**
    ```bash
    jupyter lab
    ```
